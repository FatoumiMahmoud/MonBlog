# MonBlog
Une application spring boot pour le backend d'un blog

## Install le back end 
cd MonBlogApi
mvn clean install

## Run backend
### Using maven (require a maven installation)
mvn spring-boot:run
### Using java (require JDK 8 or later)
java -jar target\monblogapi-0.0.1-SNAPSHOT.jar
